1、查看node版本
node -v
2、全局安装webpack3.6.0版本
npm install webpack@3.6.0 -g
3、局部安装webpack
cd 对应目录
npm install webpack@3.6.0 --save-dev
（--save-dev是开发时依赖，项目打包后不需要继续使用）
4、安装VUE
npm install vue --save
5、安装VUE脚手架VUE CLI3
npm install -g @vue/cli
6、使用脚手架CLI2
npm install -g @vue/cli-init
7、创建项目脚手架3
vue create 项目名
8、打开脚手架3图形化界面创建和管理项目
vue ui
9、创建项目脚手架2
vue init webpack 项目名
10、安装vue-router
npm install vue-router --save
11、安装vuex
npm install vuex
12、安装axious
npm insatll axios --save
